What is Mini-Commander?

This GNOME applet adds a command line to the panel. It features command completion, command history, changeable macros and an optional built-in clock.

Because of the changeable macros you can use Mini-Commander for many different tasks. You can simply start a program (or a short macro) or view a web page or search for a man/info page etc.

Mini-Commander has a handler on the left side which allows you to detach the applet from the panel. This feature is especially usefull for people who use vertical panels.
