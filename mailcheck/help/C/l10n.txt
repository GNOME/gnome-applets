SCREENSHOT INSTRUCTIONS
------------------------

mailcheck_applet.png

1. Create an edge panel by right-clicking on the Menu Panel and choosing Create Panel -> Edge panel.
2. Right-click on the Edge panel and choose Add to Panel -> Internet -> Inbox Monitor.
3. Take a screenshot of the desktop.
4. Use a crop tool to crop the screenshot to include only the Inbox Monitor applet.
5. Save the file in the PNG format.
