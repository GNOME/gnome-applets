1. Create an edge panel by right-clicking on the Menu Panel and choosing Create Panel -> Edge panel.
2. Right-click on the Edge panel and choose Add to Panel -> Accessories -> Stock Ticker.
3. If your computer is behind a firewall, choose Applications -> Desktop Preferences -> Network Proxy
   and specify a proxy server to use. 
4. Use GIMP to take a screenshot of the desktop.
5. In GIMP, use the crop tool to crop the screenshot to include only the Stock Ticker applet. Include the arrow
   buttons to the left and right of the scrolling display in the final screenshot.
6. Save the file in the PNG format.
