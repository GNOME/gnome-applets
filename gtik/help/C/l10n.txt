SCREENSHOT INSTRUCTIONS
------------------------

gtik_applet.png

1. Add the applet to a panel: right-click on the panel then choose Add to Panel -> Accessories -> Stock Ticker.
2. If your computer is behind a firewall, use your preference tools to specify the network proxy server for Internet connections.
3. Take a screenshot of the desktop.
4. Use the crop tool to crop the screenshot to include only the Stock Ticker applet. Include the arrow
   buttons to the left and right of the scrolling display in the final screenshot.
5. Save the file in the PNG format.
